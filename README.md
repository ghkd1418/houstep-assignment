# 2024년 (주) 두번째 프론트엔드 개발자 채용 과제

## Getting Started

First, run the development server:

1. Install packages

```bash
yarn
```

2. Run development server

```bash
yarn dev
yarn server
```

3. Open browser!

[http://localhost:3000](http://localhost:3000) 



## Stack

- Next.js
- Typescript
- React
- Mobx
- Emotion
- React-Query
